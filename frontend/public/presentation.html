GhostFund

Privacy Gas Relayer for BNB Chain

Built by Priyanshu Dangi

The Problem

Every wallet transfer is permanently public

// What the blockchain reveals:
Binance Hot Wallet → 0xFreshWallet

// Anyone can see this on BscScan — forever.
// Employers, adversaries, analytics firms
// can trace your identity instantly.

The moment you withdraw from a centralized exchange to a fresh wallet, your real identity is permanently linked to that address on-chain.

Why Now

Existing solutions don't work

Feature Tornado Cash GhostFund
Amount flexibility ✕ Fixed denominations ✓ Any BNB amount
Compliance ✕ OFAC sanctioned ✓ PPOI built-in
Token support ✕ One pool per denomination ✓ BNB + any BEP-20
Gas for recipient ✕ User's problem ✓ Relayer pays gas
Target chain ✕ Ethereum only ✓ BNB Chain native

There is no production-ready privacy relayer on BNB Chain today.

The Solution

Break the link with zero-knowledge proofs

🔒
Shield Wallet A deposits BNB into an encrypted Merkle tree
→
👀
Private Pool Tokens are hidden among all other depositors
→
⚡
ZK Proof Groth16 proof proves ownership without revealing source
→
🎉
Unshield Wallet B receives BNB — no link to Wallet A
// What the blockchain sees:
TX 1: 0xSource → RailgunProxy   // one of many deposits
TX 2: GhostRelayer → RailgunProxy   // unshield to 0xDest

// Result: No transaction between 0xSource and 0xDest ✓
